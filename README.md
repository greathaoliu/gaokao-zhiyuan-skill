# gaokao-zhiyuan-skill

![高考志愿 Skill 海报](assets/gaokao-zhiyuan-poster.png)

一个给 Codex 使用的高考志愿填报 skill。它面向零基础考生和家长，按 2026 年真实填报场景，一步一步引导用户补齐信息、核验官方数据、建立冲稳保框架，并在信息不完整时拒绝过早给出“最终方案”。

> 海报图由 Image2 生成，用于说明本 skill 的使用场景。

## 适合谁用

- 不知道高考志愿填报从哪里开始的考生和家长
- 想让 Codex 主动一问一答引导，而不是一次性填一大张表的人
- 需要按省份、年份、批次规则生成志愿方案的人
- 需要综合考虑学生兴趣、家长意见、预算、城市、专业、就业、读研和风险偏好的人

## 它会怎么工作

这个 skill 默认按 2026 年真实填报来引导。它不会一上来问一堆问题，而是默认每次只问一个最关键问题，例如：

```text
你是哪个省份的考生？
```

拿到省份后，它会继续判断下一步该问批次、选科、分数位次，还是先去查该省 2026 年官方规则。

如果用户不想回答某些偏好类问题，skill 会使用保守默认值继续推进，并明确告诉用户默认了什么。例如：

```text
我先替你默认了这些偏好，后面都可以改：
- 民办/中外合作：默认不接受高收费项目
- 风险偏好：默认中等偏稳
- 职业路径：默认本科就业和考研两手准备
```

## 主要能力

- 按省份、年份、批次识别志愿规则
- 主动查询批次线、特控线、一分一段表、投档规则等公开官方信息
- 根据分数和位次建立冲、稳、保、垫底框架
- 综合学生本人意见和家长意见
- 考虑预算、民办、中外合作、城市距离、专业偏好、就业方向、读研/考公/央国企等因素
- 检查选科、体检、单科、语种、校区、学费、大类分流、转专业、招生章程等风险
- 支持山东这类 `专业（类）+学校` 完整志愿表，也会按其他省份规则调整表格结构
- 在完整志愿表中逐项给出主要优点、主要不足、风险提示，以及近 2-3 年最低分/最低位次/计划数等录取情况

## 不会做什么

- 不保证录取结果
- 不把山东 96 个志愿规则套到全国
- 不用第三方数据库替代官方核验
- 不收集身份证号、准考证号、姓名等不必要敏感信息
- 关键信息缺失时，不输出“最终志愿表”或“可提交顺序”

## 权威信息源

最终方案应优先核验这些来源：

- 省教育考试院 / 招生考试院
- 官方志愿填报系统或官方招生计划书
- 高校本科招生官网和招生章程
- 教育部阳光高考平台
- 教育部官网、省教育厅、高校教务处/本科生院/学院官网
- 高校官方就业质量报告、本科教学质量报告、就业指导中心官网

第三方数据库只能作为筛选辅助，最终代码、计划数、学费、校区、限制条件等必须回到官方来源核验。

## 安装

这个仓库根目录就是一个完整的 Codex skill，包含：

```text
SKILL.md
agents/openai.yaml
references/
assets/
```

你可以把整个仓库下载后放到 Codex 的 skills 目录中，或直接让 Codex 从 GitHub 链接安装：

```text
请从 https://github.com/greathaoliu/gaokao-zhiyuan-skill 安装这个 Codex skill
```

安装后可以这样调用：

```text
使用 $gaokao-zhiyuan-skill 帮我做 2026 高考志愿填报
```

## 典型对话

```text
用户：使用 $gaokao-zhiyuan-skill 帮我报志愿

Codex：我会按 2026 年真实填报来引导。信息没问清前，我只给阶段性分析，不给最终志愿方案。
先问第一个关键问题：你是哪个省份的考生？
```

## 目录说明

- `SKILL.md`：skill 入口和核心工作流
- `references/guided-consultation.md`：一问一答引导、最终方案闸门
- `references/defaults-and-intake-factors.md`：用户模糊回答时的保守默认值
- `references/data-sources.md`：官方信息源和核验规则
- `references/province-year-rules.md`：省份/年份/批次规则适配
- `references/decision-factors.md`：学生、家长、专业、城市、就业等综合因素
- `references/full-volunteer-table.md`：完整志愿表输出规范
- `references/risk-checklist.md`：滑档、退档、调剂、费用、校区等风险清单
- `references/workflow.md`：零基础解释和输出模板

## 重要提醒

高考志愿填报具有强烈的省份、年份和批次差异。本 skill 只能帮助整理信息、核验规则、提示风险、生成可复核方案，不能替代考生和家长的最终决策。正式提交前，务必以省考试院官方系统、官方招生计划、高校招生章程为准。
